# Security Policy

## Reporting a Vulnerability

**Please do NOT report security vulnerabilities through public GitHub issues.**

Instead, please report them via email to **support@pdboperator.io**.

You should receive a response within 48 hours.

## Scope

This repository contains only documentation source files. For security issues
related to the PDB Operator itself, please report to the
[pdb-operator repository](https://github.com/pdb-operator/pdb-operator/security/policy).
