# Governance

For the full governance model, see the
[Governance](https://pdboperator.io/docs/community/governance) page on the
documentation site.

## Maintainers

See [MAINTAINERS.md](MAINTAINERS.md) for the current list of maintainers.

## Decision Making

- **Lazy consensus:** proposals are accepted if no maintainer objects within 72 hours
- **Voting:** simple majority for standard decisions, supermajority (2/3) for governance changes
