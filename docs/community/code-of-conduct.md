# Code of Conduct

PDB Operator follows the [CNCF Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

## Summary

We are committed to providing a welcoming and harassment-free environment for everyone, regardless of gender, gender identity, sexual orientation, disability, physical appearance, body size, race, age, or religion.

## Reporting

If you experience or witness unacceptable behavior, please report it by emailing **support@pdboperator.io**.

All reports will be handled with discretion and confidentiality.

## Enforcement

Project maintainers are responsible for enforcing the code of conduct. Actions may include warnings, temporary bans, or permanent bans from the project.
